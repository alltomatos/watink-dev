# Preferências do Projeto Watink Design System

- **Idioma padrão:** Português do Brasil (pt-BR)
- Toda documentação, comentários, textos de UI e arquivos gerados devem estar em pt-BR
- Exceções aceitas em inglês: nomes de variáveis CSS/JS, nomes de componentes (PascalCase), termos técnicos amplamente reconhecidos (token, bundle, deploy, commit, etc.)
