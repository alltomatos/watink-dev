# Flow Builder (Chatbot)

O Flow Builder é uma ferramenta visual de automação que permite criar fluxos de atendimento sem a necessidade de programação.

## Funcionalidades Principais
- **Editor Visual:** Interface drag-and-drop para construção de nós e conexões.
- **Tipos de Ações:**
    - Envio de Mensagens (Texto, Mídia).
    - Menus Interativos (Opções numéricas ou botões).
    - Transferência para Filas.
    - Chamadas de API Externas (Webhooks).
    - Captura de Variáveis (Input do usuário).
- **Triggers:** Gatilhos baseados em palavras-chave ou primeiro contato.

## Aplicações
- **Auto-atendimento:** Resolução de dúvidas frequentes sem intervenção humana.
- **Qualificação de Leads:** Coleta de informações básicas antes de passar para um vendedor.
- **Pesquisas de Satisfação:** Disparo automático de NPS após o fechamento do ticket.
